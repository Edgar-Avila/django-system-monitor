Memoria

La memoria de un computador es un dispositivo que almacena datos temporalmente. La memoria es un componente esencial en cualquier computador, ya que es la encargada de almacenar la información que necesita el procesador para realizar las operaciones. Se muestra la memoria total, la memoria disponible y la memoria usada del dispositivo.
